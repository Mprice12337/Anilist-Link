{# ── How it works ─────────────────────────────────────────────────────── #}

How it works

  • Media server → AniList: reads watched episodes and updates your AniList progress. Only moves progress forward; won't decrease a count that's higher on AniList.
  • AniList → Media server: pushes your AniList progress back into Plex/Jellyfin by marking episodes as played. Useful for initial backfill.
  • Auto-sync: disabled by default. Enable the toggle above to run automatically every 15 minutes.
  • Undo: each sync entry in the history can be individually reverted, restoring the previous AniList status and progress.
  • Requires that shows are already matched to AniList entries (run a Plex/Jellyfin metadata scan first).
