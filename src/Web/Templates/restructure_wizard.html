Library Restructuring Wizard

What this does

This wizard helps organize your anime library using AniList data. You can rename folders to match AniList titles, standardize episode filenames, or consolidate split series into a single multi-season folder.

For Plex mode, shows must already be matched to AniList (via the Plex Scan). For Local mode, folders are automatically matched by title.
